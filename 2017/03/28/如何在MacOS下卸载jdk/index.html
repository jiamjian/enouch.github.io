如何在MacOS下卸载jdk

  • 首先在终端中进入JavaVirtualMachines
    cd /Library/Java/JavaVirtualMachines
  • 执行删除 这里需要输入密码获取权限（版本号写自己的）
    sudo rm -rf jdk1.8.0_06.jdk
